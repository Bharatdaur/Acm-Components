# Hackathon Problem Statements

A React application showcasing problem statements for hackathon events with an interactive timeline component.

## Features

- Interactive problem statements display
- Timeline component for event scheduling
- Responsive design
- Modal popups for detailed problem descriptions

## Technologies Used

- React
- Vite
- CSS3
- JavaScript (ES6+)

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## Project Structure

- `src/components/ProblemStatements.jsx` - Main problem statements component
- `src/components/Timeline.jsx` - Timeline component for events
- `src/components/` - Component-specific CSS files
