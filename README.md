# ACM Components- Projects Collection

This repository contains various projects and assignments for Object-Oriented Programming (OOPs) Semester 3.

## Projects Included

### 1. Hackathon Problem Statements
A React application showcasing problem statements for hackathon events with an interactive timeline component.

#### Features
- Interactive problem statements display
- Timeline component for event scheduling
- Responsive design
- Modal popups for detailed problem descriptions

#### Technologies Used
- React
- Vite
- CSS3
- JavaScript (ES6+)

#### Getting Started
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

### 2. DSA Linked List in C++
Data Structures and Algorithms implementation focusing on linked lists using C++.

### 3. Assignment 1 & 2
Course assignments covering various OOPs concepts and implementations.

## Project Structure

- `src/components/ProblemStatements.jsx` - Main problem statements component
- `src/components/Timeline.jsx` - Timeline component for events
- `src/components/` - Component-specific CSS files
- `DSA linked list in c++` - C++ implementations
- `assignment 1` & `assignment 2` - Course assignments
